# Anil Tech Systems - Enterprise IT Solutions Website

## Project Overview
A professional, responsive, multi-page static frontend website for Anil Tech Systems, an ISO-9001 certified enterprise B2B IT services company. Designed for free Netlify hosting with future GitHub integration.

## Project Structure
```
├── index.html                     # Homepage with hero, services, case studies
├── about.html                     # Company history, values, timeline
├── contact.html                   # Netlify-ready contact form
├── services/
│   ├── system-integration.html    # System Integration service page
│   ├── datacenter.html            # Datacentre Solutions service page
│   ├── cloud.html                 # Cloud Solutions service page
│   ├── digital-transformation.html # Digital Transformation service page
│   ├── security-surveillance.html # Security & Surveillance (ATEX/PESO)
│   └── training.html              # Enterprise Training service page
├── css/
│   └── style.css                  # Complete stylesheet with design system
├── js/
│   └── main.js                    # Dropdown, hamburger, scroll effects
├── assets/                        # Images and media (add logo, hero, etc.)
├── robots.txt                     # SEO robots file
├── sitemap.xml                    # XML sitemap
└── README.md                      # Deployment instructions
```

## Key Features
- Vanilla HTML, CSS, JS (no frameworks)
- Responsive design (mobile, tablet, desktop)
- Accessible dropdown navigation with ARIA attributes
- Mobile hamburger menu with accordion services
- Netlify-ready contact form (data-netlify="true")
- WhatsApp floating button integration
- JSON-LD structured data for SEO
- Scroll reveal animations
- Animated metric counters
- Corporate color scheme (blue primary, green secondary)
